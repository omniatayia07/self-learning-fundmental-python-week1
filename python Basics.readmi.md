# ✨ ما هي لغة بايثون؟

لغة **Python** من أشهر لغات البرمجة، وتتميز بأنها:

- سهلة التعلم
- مناسبة للمبتدئين
- تُستخدم في الذكاء الاصطناعي وتحليل البيانات وتطوير المواقع
- تمتلك مجتمعًا ضخمًا ومصادر تعليمية كثيرة

---

# 🟦 1. المتغيرات (Variables)

المتغيرات تُستخدم لتخزين البيانات داخل البرنامج.

## ✅ مثال

```python
name = "Omnia"
age = 18
```

---

# 🟩 2. أنواع البيانات (Data Types)

| النوع | الوصف | مثال |
|---|---|---|
| `int` | أعداد صحيحة | `10` |
| `float` | أعداد عشرية | `3.14` |
| `str` | نصوص | `"Hello"` |
| `bool` | صح أو خطأ | `True` |

---

# 🟨 3. العمليات (Operators)

## ➕ العمليات الحسابية

```python
+
-
*
/
%
```

## ⚖️ عمليات المقارنة

```python
==
!=
>
<
>=
<=
```

## 🧠 العمليات المنطقية

```python
and
or
not
```

---

# 🟧 4. الشروط (Conditions)

تُستخدم الشروط لاتخاذ القرارات داخل البرنامج.

## ✅ مثال

```python
age = 18

if age >= 18:
    print("بالغ")
else:
    print("قاصر")
```

---

# 🔁 5. الحلقات التكرارية (Loops)

## 🔹 حلقة For

```python
for i in range(5):
    print(i)
```

## 🔹 حلقة While

```python
x = 0

while x < 5:
    print(x)
    x += 1
```

---

# 🛠️ 6. الدوال (Functions)

الدوال تساعد على تنظيم الكود وإعادة استخدامه.

## ✅ مثال

```python
def greet(name):
    print("Hello", name)

greet("Omnia")
```

## 🔹 دالة تُرجع قيمة

```python
def add(a, b):
    return a + b
```

---

# 📦 7. القواميس (Dictionaries)

القواميس تُستخدم لتخزين البيانات على شكل مفتاح وقيمة.

## ✅ مثال

```python
student = {
    "name": "Ali",
    "age": 18,
    "grade": "A"
}

print(student["name"])
```
